# MERN Todo App 📝
A full-stack **Todo Application** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.  
This app allows users to create, update, delete, and manage daily tasks efficiently with a clean and responsive UI.
---
## 🚀 Features
- Add new todos
- Edit existing todos
- Mark todos as completed
- Delete todos
- View all saved tasks
- Responsive and user-friendly interface
- RESTful API integration
